# goAI Solutions Website

Modern, story-driven marketing website for goAI Solutions — built from the [PRD](../Downloads/goai_solutions_website_redesign_prd.md) using the APDM delivery approach.

## Tech Stack

- **React 19** + **Vite**
- **Tailwind CSS v4**
- **Framer Motion** — scroll animations, loading screen, counters
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build & Deploy

```bash
npm run build
npm run preview
```

Deploy to **Vercel** — `vercel.json` is included for SPA routing.

## Sections (PRD Journey)

1. Landing — hero workflow animation + CTAs
2. The Shift — traditional delivery bottlenecks
3. The Problem — interactive challenge cards
4. Introducing APDM — model explanation
5. Human + AI — split-screen comparison
6. How goAI Is Different — comparison table
7. Real Impact — animated counters on scroll
8. Projects — expandable case study cards
9. Clients — infinite logo carousel
10. Testimonials — auto-scrolling trust signals
11. Founders — leadership profiles
12. Origin Story — timeline
13. Blogs — thought leadership cards
14. Contact — form + social links

## Dynamic Features

- Loading screen (Thinking → Delivering)
- Floating AI network background
- Scroll reveal animations (fade, slide, scale)
- Active navigation highlighting
- Back-to-top button
- Animated metric counters
